About

Draws nice boxes and arrows to your website from simple markup like (User) > (Admin)

Setup (it's easy!)

  • Put javascript links inside HTML head to jquery.js, jquery.wz_jsgraphics.js, arrowsandboxes.js

  • Place a <pre> block around your arrows-and-boxes markup and set the class to arrows-and-boxes:

    <pre class="arrows-and-boxes"> (User) > (Admin) </pre>

    Arrows-and-boxes automatically detects the code and builds nice boxes and arrows from it.

info
Different arrow types not jet implemented! Currently you only have normal lines, no arrows with directions. Will be added soon!

Basic example

Inline arrows and nodes: 

(Dennis) >likes (Food)

(Dennis) >likes (Food)
  • Define a box with "( )", and a link with ">", "<", "-".
  • An arror can be put between Boxes in a single line, or inside a box, if the target is "further away" (see below for an example)
  • Basic box format: (box_title | box_subtitle)
  • Complete box format: (box_id{custom_class}: box_title | box_subtitle >arrow_label [arrow_target1, arrow_target2, ...] <arrow2_label [arrow2_target])

Linking elswhere

Writing (node_id: My Node) gives the node the id "node_id" (must be followed by ":"). You can use it to define a target for a link inside a node definition 

(Dennis >likes [d,m])
(d:Drinks) (m:Money)

(Dennis >likes [d,m])
(d:Drinks) (m:Money)
  • Arrows between boxes: >arrow_label
  • Arrows defined INSIDE a box: >arrow_label [arrow_target1, arrow_target2, ...]

Want subtitles?

Writing a "|" after your title makes the rest of the text a subtitle. If it's too large to fit the boy, hover with your mouse to make it visible. 

(Dennis|has a subtitle)

(Dennis|has a subtitle)

Highlight nodes and arrows!

() > (boring) >> ((Highlighted))

() > (boring) >> ((Highlighted))

Use Dummy nodes to get more space

(Dennis) -likes () > (Food)

(Dennis) -likes () > (Food)

Use Dummy nodes to bend your arrows

(Dennis) - (>likes[f]) 
() (f:Food)

(Dennis) - (>likes[f]) 
() (f:Food)

HTML in your texts

(Dennis) > {{links}} (his {{Homepage}})

Custom classes for your nodes

Writing {{my_class_name}} before the ":" that is used to seperate the node id from the title will add the classname to the created node. Inspect the element if you don't trust me: 

({custom_class}:Custom class)

({custom_class}:Custom class)
